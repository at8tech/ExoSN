# ExoSN

ExoSN is an educational website that collects exercise series in Mathematics, Physics-Chemistry, and Life & Earth Sciences based on the Senegalese curriculum (from 6th grade to final year). The goal is to help students revise easily and access school resources quickly. :contentReference[oaicite:1]{index=1}

## 🧠 Features

- Downloadable exercise series in PDF format  
- Organized by subject and grade level (6th → Terminale)  
- Simple and intuitive interface  
- Free and open access  
- Responsive layout using HTML, CSS, and JavaScript :contentReference[oaicite:2]{index=2}

## 🚀 Technologies Used

- **HTML5** – page structure  
- **CSS3** – styling and layout  
- **JavaScript** – interactive elements :contentReference[oaicite:3]{index=3}

## 📚 Purpose

The main aim of this project is to create an educational platform tailored to the Senegalese science curriculum where students can find:

- In-depth exercise series  
- Organized content by subject  
- PDF resources for practice  
- Future plans for online assessment features :contentReference[oaicite:4]{index=4}
